Tutorial de Automação CRM

Aprenda a automatizar suas buscas de contratos usando o Automa

Começar GitHub

1. Instalação do Automa

1.1 Instale a Extensão

  1. Abra o Google Chrome
  2. Acesse a Chrome Web Store - Automa
  3. Clique em "Adicionar ao Chrome"
  4. Confirme a instalação quando solicitado

1.2 Verifique a Instalação

  • O ícone do Automa deve aparecer na barra de extensões
  • Clique no ícone para abrir o dashboard

2. Criação do Workflow

2.1 Crie um Novo Workflow

  1. Abra o Automa Dashboard
  2. Clique no botão "+" (Novo Workflow)
  3. Nome: "CRM Search"
  4. Descrição: "Busca de contratos no CRM"

2.2 Estrutura do Workflow

Workflow
├── Input Block (Entrada de Contratos)
├── Validation Block (Validação)
├── Loop Block (Processamento)
│   ├── Navigation Block (Navegação)
│   ├── Extract Block (Extração)
│   └── Save Block (Salvamento)
└── Export Block (Exportação)

3. Configuração dos Blocos

3.1 Bloco de Entrada

Configurações:
  • Tipo: Textarea
  • Label: "Digite os números dos contratos"
  • Placeholder: "Um contrato por linha" 
// Validação de entrada
function validateInput(input) {
    const contracts = input.split('\n');
    return contracts.every(c => /^\d{4,6}$/.test(c.trim()));
}

3.2 Bloco de Navegação

Configurações:
  • URL: Sistema CRM + Número do Contrato
  • Timeout: 30000ms
  • Wait Until: networkidle0

3.3 Bloco de Extração

Seletores:
// Seletores para extração
{
    "cod": "text:contains('COD:')",
    "banco": "text:contains('BANCO')",
    "status": ".status-label"
}

Dicas Importantes:

  • Sempre teste com poucos contratos primeiro
  • Use delays apropriados entre requisições
  • Faça backup dos resultados regularmente
  • Monitore o consumo de recursos
