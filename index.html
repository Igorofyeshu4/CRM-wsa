Recursos Principais

Busca Automática

Processe múltiplos contratos simultaneamente com validação automática.

Exportação Flexível

Exporte resultados para CSV ou integre diretamente com Google Sheets.

Segurança

Validação de dados e tratamento de erros integrados.

Como Começar

1. Instalação

Instale a extensão Automa do Chrome Web Store e acesse sua conta no sistema CRM.

2. Importação

Baixe e importe o arquivo workflow.json para o Automa.

3. Configuração

Configure suas credenciais e preferências no workflow.

4. Execução

Execute o workflow e monitore os resultados.
